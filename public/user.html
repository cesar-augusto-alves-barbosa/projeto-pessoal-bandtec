Se você ainda é iniciante na programação confira algumas informações importantes

Se você está iniciando no mundo da programação provavelmente vai começar a entender apenas algumas linhas de codigo. Sendo assim, recomendo começar a entender uma linguagem que não precise instalar muitos softwares para não confundir muito, então o javascript é uma opção. Ele não precisa ser instalado pois os principais navegadores rodam em javacript. Você pode escrever em um bloco de notas mesmo uma pagina html e css em conjunto com javascript começado a entender algumas logicas e calculos. informações.


É importante resaltar que HTML e CSS não são linguagens de programação. HTML é uma linguagem de marcação de texto que apenas encapsula dados e descreve o que fazer com eles, não como fazer e CSS é apenas uma linguagem de estilização. Já o javascript sim é uma linguagem de programação pois ele é capaz de fazer cálculos, mudar informações contidas em algum tipo de memória, tomar decisões e mudar o fluxo de execução. Porém, a programação não fica presa só ao desenvolvimento web. Então não recomendo estudar apenas sobre html, javascript e até css pois você pode gostar ainda mais de programar com outras linguagens desenvolvendo outras aplicações como softwares para computadores ou a aplicativos em mobile. Mas, se quiser aprender um pouco mais de desenvolvimento web, de uma olhada no tópico abaixo.

Quer saber mais sobre desenvolvimento web?

Você sabe o que é Front-end e Back-end?

É importante entender como funciona a web para saber o que diferencia quem faz front-end do back-end:

Podemos classificar Front-end como a parte visual de um site, aquilo que conseguimos interagir. Quem trabalha com Front End é responsável por desenvolver por meio de código uma interface gráfica, normalmente com as tecnologias base da Web (HTML, CSS e JavaScript). Algumas pessoas podem confundir um pouco esse trabalho com o que um designer faz (no passado existia uma entidade chamda Webmaster que fazia tudo isso e mais um pouco), mas a diferença aqui é que o designer vai utilizar alguma ferramenta visual para desenhar a interface, do Photoshop ao Sketch, e quem faz front-end estará mais próxima do código em si, que irá rodar em um navegador Web como Chrome, Firefox ou Safari

E o Back-end?

Back End, como o próprio nome sugere, vem da ideia do que tem por trás de uma aplicação. Pode ficar meio abstrato em um primeiro momento, mas pense que para conseguir usar o Facebook no dia a dia, os dados do seu perfil, amigos e publicações precisam estar salvos em algum lugar, sendo esse lugar um banco de dados e processados a partir de lá. Não basta apenas o front-end em HTML e CSS! O Back End trabalha em boa partes dos casos fazendo a ponte entre os dados que vem do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias em um ambiente onde o usuário final não tenha acesso e possa manipular algo.

Se você já sabe um pouco de programação confira algumas dicas e sugestões

Dicas para um bom programador

  • 1)Nunca pare de estudar: como a programação é uma das áreas que envolve a tecnologia, que se altera ou evolui constantemente, é preciso estar por dentro de tudo.
  • 2)Sempre coloque seu código no Github: GitHub é um sistema de gerenciamento de projetos e versões de códigos assim como uma plataforma de rede social criado para desenvolvedores. Mas para que o GitHub é utilizado? Entre outras coisas, ele permite que você trabalhe em projetos colaborativos com desenvolvedores de todo o mundo, planeje seus projetos e acompanhe o trabalho. Então, mesmo que você sejá iniciante, colocar o seu codiogo no GitHub ajúda a você interagir mais com a comunidade de programadores.
  • 3)Programar significa escrever código: Quando se esta estudando programação é necessario escrever o codigo mesmo, não só ler livros, artigos, etc. Não é que eles não ajudam, porem, ficar apenas estudando por meio deles não vai te fazer evoluir. Quando se esta estudando escrevendo linhas de codigos você pode ver se o que estudou surtiu efeito e se acontecer algum erro você vai saber aprender com ele.

Confira algumas estenções do Visual Studio Code

Se você utiliza o escritor de codigo fonte Visual Studio Code confira algumas estenções que te auxiliarão no desenvolvimento web

  • Auto rename tag: Quando queremos trocar um nome de uma tag temos que alterar em dois lugares, mas, com essa extenção só precisamos alterar em um lugar só.
  • Code Runner: Executa snippets de código ou o arquivo de código para varias linguagens.
  • CSS Peek: Utilizando um atalho segurando a tecla ctrl com o mouse em cima de uma classe é possivel visualizar rapidamento como está sendo estilizado um elemento.
  • Themes: Existem muitos temas para o VS Code, basta escolher um, instalar e pronto. Eu utilizo o Dracula Pure.
  • Icons: Os icones no VS Code, além de estiliza-lo, te ajuda a identificar mais rapido que tipo de arquivo ou o tipo de conteudo de uma pasta esta em certo local. Eu utilizo o Material Icon Theme.
